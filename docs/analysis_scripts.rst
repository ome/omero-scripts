

Analysis scripts
================

Kymograph
---------

The Kymograph script uses Lines or Polylines drawn on a source timelapse image to
create new kmograph images.

.. automodule:: Kymograph
   :members:


Kymograph Analysis
------------------

The Kymograph Analysis script uses can analyse the tracks from a Kymograph image to
caculate the velocity of objects. It uses Lines or Polylines manually drawn on to
the Kymograph image, and creates a csv table output.

.. automodule:: Kymograph_Analysis
   :members:
